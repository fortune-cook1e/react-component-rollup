# react-component-rollup

rollup 打包的 react 组件

## Getting Started

```javascript
yarn dev:story // for storybook dev
yarn dev:page // for page dev
yarn build:page // build for page
yarn build:component // build for component
yarn watch:component // dev for component
yarn build:doc
yarn pre-publish
```
